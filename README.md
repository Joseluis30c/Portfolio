# 🧠 Descripción

**Portfolio** es un repositorio que reúne diferentes proyectos desarrollados como parte de mi aprendizaje y experiencia en desarrollo de software.

El objetivo es mostrar ejemplos reales de:

- Aplicaciones backend
- Automatización
- Integración con APIs
- Uso de inteligencia artificial
- Desarrollo con .NET

Cada proyecto dentro del portfolio representa una solución a un problema específico o una demostración de una tecnología.

------------------------------------------------------------------------

# 🎯 Objetivo del proyecto

Los principales objetivos de este repositorio son:

- Mostrar mis **habilidades técnicas como desarrollador**
- Centralizar mis proyectos más importantes
- Servir como referencia para futuros desarrollos
